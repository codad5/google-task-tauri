
# GOOGLE TASK DESKTOP APP

## DESCRIPTION
A light weight desktop app for google task built with tauri, react, rust and typescript. 

## INSTALLATION

```bash
git clone
cd google-task-tauri
npm install
npm run launch:dev
```
